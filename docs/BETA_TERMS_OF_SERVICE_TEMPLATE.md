## 2. Beta Service Description

LexiCraft is an educational platform that allows children to earn money by learning English vocabulary. The platform operates on a deposit-based model:

1. **Parent deposits funds** (NT$500 - NT$1,000 during beta)
2. **Child learns vocabulary** through interactive lessons
3. **Child earns points** by completing verification tests
4. **Parent withdraws cash** when child earns points
5. **Parent decides how to reward** the child (allowance, toys, savings, etc.)

**Beta Limitations:**
- Maximum deposit per family: NT$1,000 during beta period
- Service may experience interruptions or downtime

## 5. Learning and Earning Points

### 5.1 Learning Process

- Children learn vocabulary through interactive flashcards
- Daily word limit: 20 words per day (during beta)
- Verification tests: 6-option multiple choice questions
- Points are earned upon successful completion of verification tests

### 5.2 Point System

- **1 point = NT$1** (1:1 conversion ratio)
- Points are earned when child passes verification tests
- Points are added to available balance after verification
- Points can be withdrawn as cash by parent

### 5.3 Verification System

- **Day 1:** Learn words + immediate test
- **Day 3:** Retention test
- **Day 7:** Final verification test → Points unlock
- Points are only unlocked after successful completion of all tests
